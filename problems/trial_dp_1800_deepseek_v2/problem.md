# 最大异或划分

**时间限制：** 1 秒  
**内存限制：** 256 MB  

给定一个长度为 $n$ 的整数数组 $a$。你需要将数组划分为若干段**非空连续子数组**。每个子数组的得分为该子数组所有元素的按位异或（XOR）值。一个划分方案的总得分为其所有子数组得分之和。

请在所有合法划分方案中，求出**最大可能的总得分**。

## 输入格式

第一行包含一个整数 $n$（$1 \le n \le 5000$）—— 数组的长度。

第二行包含 $n$ 个整数 $a_1, a_2, \ldots, a_n$（$-10^9 \le a_i \le 10^9$）—— 数组的元素。

## 输出格式

输出一个整数，表示可达到的最大总得分。

## 样例

### 样例 1

**输入：**
```
4
1 2 3 4
```

**输出：**
```
10
```

**样例解释：**
所有元素均为非负数。由于任意一组非负整数的异或值不会超过它们的和，因此最优策略是将每个元素单独作为一个子数组。总得分为 $1 + 2 + 3 + 4 = 10$。

### 样例 2

**输入：**
```
3
-3 -1 2
```

**输出：**
```
4
```

**样例解释：**
最优划分为 $[-3, -1]$ 和 $[2]$。
- $[-3, -1]$ 的异或值：$(-3) \oplus (-1) = 2$
- $[2]$ 的异或值：$2$

总得分：$2 + 2 = 4$。

（若所有元素各自独立，得分为 $(-3) + (-1) + 2 = -2$，更差。）

### 样例 3

**输入：**
```
4
-5 -7 3 1
```

**输出：**
```
6
```

**样例解释：**
最优划分为 $[-5, -7]$、$[3]$、$[1]$。
- $[-5, -7]$ 的异或值：$(-5) \oplus (-7) = 2$
- $[3]$ 的异或值：$3$
- $[1]$ 的异或值：$1$

总得分：$2 + 3 + 1 = 6$。

（全部各自独立得分为 $(-5) + (-7) + 3 + 1 = -8$。）

## 提示

答案可能超出 32 位整数的范围，请使用 64 位整数（例如 C++ 中的 `long long`）。
