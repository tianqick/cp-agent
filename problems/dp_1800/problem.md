# Bounded Difference Subsequence

## 题目描述

给定一个长度为 $n$ 的整数序列 $a_1, a_2, \dots, a_n$ 和一个整数 $k$。

你需要找出该序列的一个最长子序列（不必连续），使得子序列中任意相邻两个元素（在子序列中的相邻）在原序列中的下标递增，并且它们的差的绝对值不超过 $k$。

换句话说，如果子序列的下标为 $i_1 < i_2 < \dots < i_m$，那么对于所有 $1 \le j < m$，都有 $|a_{i_{j+1}} - a_{i_j}| \le k$。

输出这个最长子序列的长度。

## 输入格式

第一行包含两个整数 $n$ 和 $k$（$1 \le n \le 2 \times 10^5$, $0 \le k \le 10^9$）。

第二行包含 $n$ 个整数 $a_1, a_2, \dots, a_n$（$1 \le a_i \le 10^9$）。

## 输出格式

输出一个整数，表示满足条件的最长子序列的长度。

## 样例输入1

```
5 2
3 1 4 1 5
```

## 样例输出1

```
3
```

## 样例解释1

一个可行的最长子序列是 $[1, 3, 5]$（取下标 2, 1, 5，但下标必须递增，所以是 $[1, 4, 5]$（下标 2, 3, 5）或者 $[3, 1, 1]$（下标 1, 2, 4）等。实际上，取 $[1, 1, 5]$（下标 2, 4, 5）满足条件：$|1-1|=0 \le 2$, $|5-1|=4 > 2$，不行。取 $[3, 1, 1]$（下标 1, 2, 4）：$|3-1|=2 \le 2$, $|1-1|=0 \le 2$，长度为 3。也可以取 $[1, 4, 5]$（下标 2, 3, 5）：$|1-4|=3 > 2$，不行。所以答案是 3。

## 样例输入2

```
4 0
5 5 5 5
```

## 样例输出2

```
4
```

## 样例解释2

所有元素都相同，任意子序列都满足条件，最长就是整个序列。

## 数据范围

- 对于 $30\%$ 的数据，$n \le 1000$。
- 对于 $100\%$ 的数据，$1 \le n \le 2 \times 10^5$, $0 \le k \le 10^9$, $1 \le a_i \le 10^9$。
