# 数组求和

## 题目描述

给定一个长度为 $n$ 的正整数数组 $a_1, a_2, \ldots, a_n$，请计算所有元素的和。

## 输入格式

第一行一个正整数 $n$，表示数组长度。

第二行 $n$ 个正整数 $a_1, a_2, \ldots, a_n$，表示数组元素。

## 输出格式

输出一个整数，表示数组所有元素的和。

## 样例

**样例输入 1：**
```
5
1 2 3 4 5
```

**样例输出 1：**
```
15
```

**样例输入 2：**
```
1
42
```

**样例输出 2：**
```
42
```

## 样例解释

样例 1：$1 + 2 + 3 + 4 + 5 = 15$

## 数据范围

- $1 \leq n \leq 10^5$
- $1 \leq a_i \leq 10^6$
