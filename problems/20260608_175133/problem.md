# Range GCD Queries

## 题目描述

给定一个长度为 $n$ 的数组 $a_1, a_2, \ldots, a_n$。你需要处理 $q$ 个操作，操作有两种类型：

1. **区间加法**：`1 l r x` — 将区间 $[l, r]$ 内的所有元素加上 $x$
2. **区间GCD查询**：`2 l r` — 查询区间 $[l, r]$ 内所有元素的最大公约数

**注意**：数组下标从 $1$ 开始。

## 输入格式

第一行包含两个整数 $n$ 和 $q$ $(1 \le n, q \le 2 \times 10^5)$

第二行包含 $n$ 个整数 $a_1, a_2, \ldots, a_n$ $(1 \le a_i \le 10^{12})$

接下来 $q$ 行，每行一个操作：
- `1 l r x` — 区间加法 $(1 \le l \le r \le n, |x| \le 10^6)$
- `2 l r` — 区间GCD查询 $(1 \le l \le r \le n)$

## 输出格式

对于每个类型 $2$ 的操作，输出一行表示区间 $[l, r]$ 的GCD值。

## 样例

### 输入
```
5 5
4 8 12 6 10
2 1 3
2 2 4
1 1 4 2
2 1 3
2 2 5
```

### 输出
```
4
2
2
2
```

### 解释
- 初始数组：$[4, 8, 12, 6, 10]$
- 查询 $[1, 3]$：$\gcd(4, 8, 12) = 4$
- 查询 $[2, 4]$：$\gcd(8, 12, 6) = 2$
- 区间加法：$[1, 4]$ 每个元素加 $2$，数组变为 $[6, 10, 14, 8, 10]$
- 查询 $[1, 3]$：$\gcd(6, 10, 14) = 2$
- 查询 $[2, 5]$：$\gcd(10, 14, 8, 10) = 2$

## 数据范围

- $1 \le n, q \le 2 \times 10^5$
- $1 \le a_i \le 10^{12}$
- $|x| \le 10^6$
- $1 \le l \le r \le n$

## 算法提示

关键观察：$\gcd(a, b) = \gcd(a, b - a)$。利用差分数组可以将区间GCD查询转化为维护差分数组的GCD。同时需要使用线段树来支持单点修改和区间GCD查询，以及树状数组来快速获取某个位置的值。

时间限制：3秒
内存限制：256MB
