# 最优分组

## 题目描述

有 $n$ 个学生，第 $i$ 个学生的考试成绩为 $a_i$。现在需要将他们分成若干个小组，满足以下条件：

1. 每个小组最多有 $k$ 个学生。
2. 每个小组中，成绩最高的学生和成绩最低的学生的成绩之差不超过 $d$。

每个学生恰好属于一个小组。请问最少需要分成多少个小组？

## 输入格式

第一行包含三个正整数 $n, k, d$，分别表示学生数量、每组最多人数和成绩差上限。

第二行包含 $n$ 个正整数 $a_1, a_2, \ldots, a_n$，表示每个学生的成绩。

## 输出格式

输出一个正整数，表示最少需要的小组数量。

## 样例输入

```
7 3 2
6 1 5 2 1 5 3
```

## 样例输出

```
3
```

## 样例解释

将学生按成绩排序后为 $[1, 1, 2, 3, 5, 5, 6]$。

- 第一组：$[1, 1, 2]$，成绩差 $2 - 1 = 1 \leq 2$，共 $3$ 人。
- 第二组：$[3, 5, 5]$，成绩差 $5 - 3 = 2 \leq 2$，共 $3$ 人。
- 第三组：$[6]$，共 $1$ 人。

共需 $3$ 个小组。

## 数据范围

- $1 \leq n \leq 10^5$
- $1 \leq k \leq n$
- $0 \leq d \leq 10^9$
- $1 \leq a_i \leq 10^9$

## 时间限制

2 秒

## 内存限制

256 MB
