# Make It Increasing

## 题目描述

给定一个长度为 $n$ 的整数数组 $a_1, a_2, \ldots, a_n$。

你可以执行以下操作**任意次**：
- 选择一个下标 $i$（$1 \le i \le n$），将 $a_i$ 的值加 $1$。

你的目标是使数组**严格递增**，即满足 $a_1 < a_2 < \cdots < a_n$。

请计算最少需要多少次操作。

## 输入格式

第一行包含一个整数 $n$（$1 \le n \le 1000$）—— 数组的长度。

第二行包含 $n$ 个整数 $a_1, a_2, \ldots, a_n$（$1 \le a_i \le 10^6$）—— 数组的元素。

## 输出格式

输出一个整数 —— 使数组严格递增所需的最少操作次数。

## 样例

### 输入 #1
```
3
1 2 3
```

### 输出 #1
```
0
```

### 输入 #2
```
3
2 1 3
```

### 输出 #2
```
3
```

### 输入 #3
```
5
1 1 1 1 1
```

### 输出 #3
```
10
```

## 样例解释

- 样例 1：数组已经是严格递增的，不需要任何操作。
- 样例 2：最优方案是变为 `2 3 4`。$a_2$ 需要增加 $2$ 次（从 $1$ 变为 $3$），$a_3$ 需要增加 $1$ 次（从 $3$ 变为 $4$），总计 $0 + 2 + 1 = 3$ 次操作。
- 样例 3：最优方案是变为 `1 2 3 4 5`，需要 $0 + 1 + 2 + 3 + 4 = 10$ 次操作。
